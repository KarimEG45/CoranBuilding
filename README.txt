The Coran Building - Dashboard de Mémorisation
===============================================

INSTALLATION ET LANCEMENT
-------------------------
1. Pas d'installation nécessaire.
2. Double-cliquez simplement sur `QuranBuildingApp.exe`.
3. L'application démarrera et lancera le serveur. Ouvrez votre navigateur à l'adresse http://localhost:8001.

FONCTIONNALITÉS
---------------
- Tableau de bord visuel de votre progression.
- Intégration de l'IA pour la vérification de récitation.
- Gestion multi-utilisateurs via le menu Paramètres.
- Section Inspirations pour la motivation.

FICHIERS
--------
- QuranBuildingApp.exe : Le programme principal.
- coatings/ : Dossier des enregistrements (créé automatiquement).
- quran_app.db : Base de données de votre progression (créée automatiquement).

NOTE TECHNIQUE
--------------
L'application est autonome. Aucune fenêtre de commande ne s'affichera (mode silencieux).
Pour arrêter l'application, vous pouvez fermer le processus via le Gestionnaire des tâches ou simplement fermer l'onglet du navigateur (le serveur continuera en arrière-plan jusqu'à l'arrêt du PC ou fermeture forcée).
